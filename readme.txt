== Frequently Asked Questions ==

**Will I need to configure the plugin?**

Yes, after installing the plugin you have add some slides, Then you have to write a shortcode called [dr-info-slider] where you want to show the output of the plugin .

== Installation and Use ==

1. Upload the entire `dr-info-slider` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Dcotor** menu in your WordPress admin screen.

= Basic Usage
Step 1: Install the plugin
Step 2: Activate the plugn
Step 3: Create some slide from Doctor Section
Step 4: Use this shortcode [dr-info-slider][/dr-info-slider] where you want to show the output of the slider
